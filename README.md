tmuxbro
=======

tmuxbro is a ssh multiplexer using <code>tmux</code> as its backend, and can be used as alternative of <code>omnitty</code>.

<pre>
$ cat /path/to/host_list
root@10.210.201.81
root@10.210.201.82
root@10.210.201.83
...

$ ./tmuxbro.sh /path/to/host_list
</pre>
